# **App Name**: WanderWise

## Core Features:

- Personalized Itinerary Generator: Generates customized travel itineraries based on user preferences, budget, and travel style.
- AI Travel Assistant: An AI-powered travel assistant that can answer questions, provide recommendations, and book travel arrangements.
- Multilingual Phrasebook: Offer real-time translation for common phrases and scenarios.
- Augmented Reality Landmark Information: Overlay historical and cultural information onto real-world landmarks using the device's camera.
- Social Travel Platform: A platform for users to share their travel experiences, photos, and recommendations with others.
- Interactive Map: Visually display attractions, restaurants and hotels.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to evoke a sense of wanderlust and adventure.
- Secondary color: Sandy beige (#F5F5DC) for a warm and inviting feel.
- Accent color: Sunset orange (#FFA500) for CTAs and highlights.
- Font: 'Lato' for a clean, modern, and readable experience.
- Use travel-themed icons (airplanes, maps, landmarks) with a minimalist design.
- Card-based layout with large, high-quality images to showcase travel destinations.
- Subtle parallax scrolling effects to add depth and visual interest.
